5. HTML Classes (Back to top)

Using inline CSS can be repetitive and annoying. Imagine that you need to create ten div and then apply the same css in those. There's an easiest way to do it using classes. Classes are a reusable piece of code that helps you applying the same customization in different places in your code. Define a class in the div and then reference this class in the style scope, with all the attributes described inside of it.
Classes are represented with . in the CSS style.

6.HTML ID's (Back to top)

We have already defined ID's in this page, using the "index" and "back to top" links. ID's should be unique, it's like yours driver license, you can't have more than one.
ID's are represented with # in the CSS Style.
